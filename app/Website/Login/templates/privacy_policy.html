Privacy Policy

Last updated: Feburary 4, 2026

This Privacy Policy explains how Osmo (“the Application”, “we”, “us”) accesses and uses information when you choose to connect third-party services such as Google Calendar and Canvas LMS.

1. Overview

The Application is designed to help students view academic assignments and optionally synchronize them with a personal calendar. We prioritize data minimization and only access information strictly necessary to provide core functionality. We do not sell, rent, or monetize user data.

2. Information We Access

2.1 Google Calendar Data

If you connect a Google Calendar account, the Application uses Google OAuth to request the minimal scopes needed to create and update assignment events inside a dedicated Osmos Tracker calendar that is associated with your user account.

Data accessed includes:

  • Calendar metadata required to create or select the Osmos Tracker calendar (calendar ID and timezone).
  • Event fields derived from your assignments, such as titles, descriptions, and due dates/times.

The Application does not access:

  • Gmail, Google Drive, Google Classroom, or other Google products.
  • Events outside of the Osmos Tracker calendar that you explicitly authorize.

2.2 Canvas LMS Data

If you connect a Canvas LMS account, the Application accesses read-only data via the Canvas API:

  • Course names
  • Assignment titles
  • Assignment due dates

The Application does not access:

  • Grades
  • Submissions
  • Messages
  • Files
  • Personal communications

Canvas is a product of Instructure. This project is not affiliated with or endorsed by Instructure.

3. How Information Is Used

Accessed data is used solely to:

  • Display courses and assignments inside the Application.
  • Allow users to optionally create Google Calendar events for assignments.

Data is not used for:

  • Advertising
  • Analytics
  • Profiling
  • Marketing
  • Resale or sharing with third parties

4. Data Storage and Retention

4.1 Canvas LMS

If the user connects Canvas LMS, Canvas course and assignment metadata (such as course names, assignment titles, and due dates) is stored within the Application for the purpose of displaying and managing assignments in the tracker. The Application does not store grades, submissions, messages, files, or personal communications from Canvas.

4.2 Google Calendar

If the user connects Google Calendar, data is accessed only after user consent and is not permanently stored. Stored assignment data is retained only for as long as the user maintains an active account or connected Canvas integration and is deleted when the user disconnects the integration or removes their account.

5. Data Sharing

We do not share user data with third parties.

Data obtained through Canvas LMS and Google APIs are used only to perform actions explicitly requested by the user within the Application.

6. User Consent and Control

  • All third-party access requires explicit user consent.
  • Users may revoke Google account access at any time via their Google Account settings.
  • Canvas API tokens can be revoked directly from the user's Canvas account.
  • The Application respects all revocations immediately.

7. Data Security

Reasonable technical and organizational safeguards are implemented to protect authorized data from unauthorized access, disclosure, or misuse.

8. Google API Compliance

The Application’s use of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.

9. Children’s Privacy

The Application is not directed toward children under the age of 13 and does not knowingly collect personal information from children.

10. Changes to This Policy

This Privacy Policy may be updated from time to time.

Material changes will be reflected by updating the effective date above.
